# License Boundaries — Finance Integrations

## Policy

- AgentSpine does not vendor or copy OpenBB source code.
- OpenBB is consumed only as an optional external dependency.
- Finance plugin code must remain provider-agnostic at contract level.

## Operator caution

- Verify OpenBB and upstream data-source license terms before production use.
- Do not assume SaaS redistribution rights from dependency presence alone.
- Keep provider attribution and usage logs for compliance/audit review.
